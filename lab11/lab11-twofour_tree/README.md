This is my implementation of a 2-4 Tree in Java!

TwoFourTree.java contains the node and tree logic, including insertion, splitting, and inorder traversal.
TwoFourTreeDriver.java tests the tree by inserting values that trigger node splits.

How to run:
1. Compile both files:
   javac TwoFourTree.java TwoFourTreeDriver.java
2. Run the driver:
   java TwoFourTreeDriver

The program will insert values and print the inorder traversal.
