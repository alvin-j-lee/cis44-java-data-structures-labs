# AVL Tree Implementation

This is my AVLTree.java program! It implements a self-balancing binary search tree using AVL rotations.

The tree maintains balance using:
- Left and right rotations
- Double rotations (LR and RL)
- Height tracking and balance factors

To run:
1. Compile: javac AVLTree.java AVLTreeDriver.java
2. Run: java AVLTreeDriver

The driver inserts values that trigger all rotation cases and prints traversals.
