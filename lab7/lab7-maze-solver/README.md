This is the directory for lab7-maze-solver.

The program solves a 2D maze using recursion and backtracking.

The maze is represented as a grid of characters:
- `S` = Start
- `F` = Finish
- `#` = Wall
- ` ` = Open path

The solver recursively explores the maze from the start position, marking visited cells and backtracking 
when a dead end is reached until a path to the finish is found.
